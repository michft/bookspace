# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-XX-XX

### Added
- Initial alpha release
- Workspace-based bookmark organization
- Automatic workspace detection via Firefox containers
- Manual workspace switching via popup
- Support for `bookspace-none` and `bookspace-all` modes
- Container Info display in popup
- Clickable container tags for workspace switching
- 3-column landscape popup layout (1200px wide)
- Debug state and API availability displays

### Changed
- Popup layout converted from portrait (320px) to landscape (1200px) with 3 columns

### Fixed
- Workspace transition logic - bookmarks now properly moved back to bookspace folder before displaying next workspace

### Known Issues
- This is ALPHA software - things may be broken
- Not ready for production use
