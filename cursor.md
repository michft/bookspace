# Cursor IDE Notes

## Project Context

Bookspace is a web extension for Zen Browser that organizes bookmarks by workspace using folders.

## Development Notes

- Extension uses Manifest V2 (compatible with Firefox/Zen Browser)
- Main files:
  - `background.js` - Core extension logic, workspace switching, bookmark management
  - `popup.html` / `popup.js` - Extension popup UI
  - `manifest.json` - Extension manifest

## Common Tasks

- Testing workspace transitions
- Verifying container detection
- Checking bookmark organization

## Reminders

- This is ALPHA software (version 0.0.1)
- Things may be broken
- Not ready for production use
